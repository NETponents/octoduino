What's new in this version (v0.1.3)

  • Updated PB BIOS.
  • Updated documentation.

  • Parsing of lang.oct to find target language.
Known bugs
  • Arduino does not handle running out of space on SD card or RAM.
  • SWAP folder must be cleared before each run.
  • SD card access is kind of slow (will fix in near future).
  • Any version of PBBIOS will run with any version of the bootloader (and will hit numerous errors)

If you find a bug that is not listed above, please report it in our issue tracker on GitHub at http://github.com/NETponents/octoduino/issues.
