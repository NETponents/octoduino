Supported Devices

The following devices are tested every build and will work with Octoduino:

  • Uno
  • Mega
  • Micro

The following devices have special hardware on them, so we are not able to run simulations on them. They may or may not work with Octoduino.

  • Mega ADK
  • Yun
  • Leonardo

As of the current version, Energia devices are not supported. The reason being that no usable SDFAT library exists. Hopefully this will be changed in a future version.
